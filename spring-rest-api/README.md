PUT - полностью обновляет объекта или создает при его отсутствии

PATCH - может обновлять отдельные поля объекта

При реализации PATCH-метода для объекта DTO, который бы принимал этот метод, можно разрешить иметь некоторые поля равные null.
Соответственно при таком значении поля объекта обновляться не будут.
