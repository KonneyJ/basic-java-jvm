# Жизненный цикл сборки
- mvn clean  
В ходе этого цикла происходит очистка проекта. Удаляются все ранее созданные в процессе сборки артефакты: 
jar-файлы, файлы классов и др.
- mvn compile  
В ходе этого цикла происходит компиляция проекта, без тестов. Создается папка target. В подпапке classes находятся 
скомпилированные классы. В подпапке generated-sources находится автоматически сгенерированный исходных код. В подпапке 
maven-status находится служебная информация о процессе сборки.
- mvn test  
Происходит выполнение тестов с использованием указанного тестового фреймворка. Включает в себя очистку и компиляцию кода.
В консоли указывается информация о пройденных тестах. Генерирует папки аналогичные предыдущему пункту, но для тестов 
проекта.
- mvn package  
Происходит упаковка артефакта, например в jar-файл, который создается в ходе этого цикла. Включает очистку, компиляцию 
и тесты. Также создает папку maven-archiver с файлом pom.properties.
